API¶

Generate piano keyboard¶

Construct a PianoComponentGenerator object

const piano = new PianoComponentGenerator();

The PianoComponentGenerator object could access the following functions

piano.setHeight(height)¶

Customise and set the height of the piano keyboard to height before generating it.

Arguments

  • height (Number()) – the height of the piano keyboard

piano.setWidth(width)¶

Customise and set the width of the piano keyboard to width before generating it.

Arguments

  • width (Number()) – the width of the piano keyboard

piano.makeKeyboard(oct_num)¶

Generate a piano keyboard with oct_num octaves. Only support sound for the first two octaves.

Arguments

  • oct_num (Number()) – the number of octave that the piano keyboard includes

piano.selectChord(chord)¶

Display the corresponding notes of chord on the piano keyboard by changing the color of the keys. Valid chords include C, D, E, F, G, A, B, C#, D#, F#, G#, A#.

Arguments

  • chord (String()) – the name of a chord

piano.clearChord(chord)¶

Reset the chord’s piano keys to the original color. Valid chords include C, D, E, F, G, A, B, C#, D#, F#, G#, A#

Arguments

  • chord (String()) – the name of a chord

piano.playChord(chord)¶

Produce the corresponding sound for chord. Valid chords include C, D, E, F, G, A, B, C#, D#, F#, G#, A#

Arguments

  • chord (String()) – the name of a chord

piano.makeInputField(form_id, textField_id_1, textField_id_2)¶

Generate a form element with two text fields that takes users input

Arguments

  • form_id (String()) – id of the form element

  • textField_id_1 (String()) – id of the first text field

  • textField_id_2 (String()) – id of the second text field

piano.makeStartButton()¶

Generate a start button with text “Start”

piano.playChords(chords, time_interval, text_field_id_1, text_field_id_2)¶

Take users inputs from two text fields and produce sound of the list input chords. Valid chords include C, D, E, F, G, A, B, C#, D#, F#, G#, A#

Arguments

  • chords (Array[String]()) – a list of chords

  • time_interval (Number()) – a positive number, the list of chords will finish playing within this time interval

  • text_field_id_1 (String()) – id of the first input field, usually taking a list of chords as input

  • text_field_id_2 (String()) – id of the second input field, usually taking a time interval as input

piano.playMelody(note_list, time_interval, text_field_id_1, text_field_id_2)¶

Take users inputs from two text fields and produce sound of the list input notes. Valid chords include C, D, E, F, G, A, B, C#, D#, F#, G#, A#

Arguments

  • note_list (Array[String]()) – a list of notes

  • time_interval (Number()) – a positive number, the list of notes will finish playing within this time interval

  • text_field_id_1 (String()) – id of the first input field, usually taking a list of notes as input

  • text_field_id_2 (String()) – id of the second input field, usually taking a time interval as input

Generate progress bar¶

Construct a ProgressBarGenerator object

const progress_bar = new ProgressBarGenerator();

The ProgressBarGenerator object could access the following functions

progress_bar.makeProgressBar()¶

Generate a progress bar that is invisible in the initial state

progress_bar.startProgressBar(time_interval)¶

Start showing the progress bar

Arguments

  • time_interval (Number()) – a positive number, the progress bar will end when the time interval passed

Generate chord buttons¶

Construct a selectChordButtonGenerator object

const chord_buttons = new selectChordButtonGenerator();

The selectChordButtonGenerator object could access the following functions

chord_buttons.makeButtonList(buttons)¶

Generate buttons.length buttons where each of the button is associated with a chord. By clicking the edit button, users are able to modify the associated chord.

Arguments

  • buttons (Array[String]()) – a list of chords where valid chords include C, D, E, F, G, A, B, C#, D#, F#, G#, A#
